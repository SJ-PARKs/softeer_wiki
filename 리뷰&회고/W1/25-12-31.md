# 오늘 리뷰

###

**API를 이용해서 GDP 데이터 수집**

- Scrapping이 아닌 API를 통해서 GDP 정보를 얻어 올 수 있는지 확인하였다. 이 과정에서 sdmx라는 표준을 알게 되었다.
- GDP 데이터 종류가 다양하여 원하는 데이터를 찾는 데 어려움을 겪었다. GDP 데이터를 다루기 위해서는 많은 도메인 지식과 sdmx와 같은 이해가 필요하다는 것을 알게 되었다.

- IMF 데이터를 호출하기 위한 다양한 라이브러리를 알아보았다.
- 그중 imfp라는 라이브러리를 알게 되었고, 기존 IMF 데이터 API 호출 과정에서 어떤 문제가 있었는지, 그리고 이를 어떻게 해결했는지를 중심으로 살펴보았다.

---

**ETL 프로세스 설계에 대한 고민**

- 갱신되는 상황에서 과거의 데이터를 어떻게 저장하면 좋을지에 대해 생각해보았다.
- 현재 내 데이터베이스 스키마를 ERD로 표현하였다. 표현하니 내 문제점이 보였다.
- 결과적으로 하나의 테이블을 따로 만들어 연도 데이터와 함께 차곡차곡 쌓아두는 방식이 좋을 것이라 판단하였다.

---

**SQL 학습 정리**

- SQL 공부를 진행하였다.
- window 함수에 대해 알아보았고, SELECT INTO, View, With 함수에 대해서도 정리하였다.
- 또한 조원들과 면접에서 나왔던 SQL 질문들을 복기하였다.
