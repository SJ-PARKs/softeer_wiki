# Predicting Election Results from Twitter Using Machine Learning Algorithms

## 느낀점
우선 논문에서 제시한 정확도 94%라는 수치에 대해 정말로 잘 학습된 모델인지 의문이 들었다. 실제 텍스트의 긍정과 부정을 판단할 때 단순히 긍정/부정 이분법만으로는 풍자, 비꼼, 중립적인 의견과 같은 복잡한 표현이 충분히 반영되지 못했을 가능성이 있다고 생각한다. 따라서 감성 분석의 높은 정확도가 곧바로 실제 민심을 정확히 반영한다고 보기는 어렵다고 느꼈다.

그럼에도 불구하고, SVM이라는 비교적 딥러닝보다 단순한 모델로 이러한 성능을 보였다는 점은 인상 깊었다. 이는 머신러닝 기법만으로도 사회 현상을 상당 부분 분석할 수 있음을 보여준 사례라고 생각한다. 그러나 동시에 기술이 더욱 발전할 경우, 이러한 분석 기법이 여론 조작이나 정보 왜곡에 악용되어 민주주의를 위협할 가능성도 존재한다고 느꼈다.

그래서 인공지능 기술이 발전해 나가는 과정에서, 이를 부정적으로 사용하는 것을 방지하기 위한 윤리적 기준과 제도적 안전장치가 반드시 필요하다고 생각하게 되었다. 또한 이러한 기술이 정치적 조작이나 선동이 아니라, 사회에 실질적으로 도움이 되는 방향으로 활용될 수 있도록 지속적인 연구와 관리가 이루어져야 할 것이라고 느꼈다.
